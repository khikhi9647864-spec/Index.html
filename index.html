FAQ

Câu hỏi thường gặp

Web tôi deploy có ảnh hưởng đến web khác không?

Không. Mỗi web chạy trong container Linux cách ly hoàn toàn (namespace + cgroup), không share process hay filesystem với web khác.

Cần trả phí gì để chạy 24/7?

Gói Free sleep sau 15 phút không có traffic. Nâng lên Pro ($9/tháng) để web sống 24/7 không sleep.

Hỗ trợ ngôn ngữ nào?

Node.js, Bun, Deno, Python, Go, Ruby, PHP, Rust, Static HTML — hoặc bất kỳ Docker image nào.

Auto deploy hoạt động thế nào?

Sau khi kết nối GitHub, mọi commit push vào branch được chọn sẽ trigger build và deploy tự động. Mỗi PR có preview URL riêng.

Có backup database không?

Có. PostgreSQL managed backup hằng ngày, giữ 7 ngày (Free), 30 ngày (Pro), 90 ngày (Team).

Sẵn sàng đưa web lên internet?

Miễn phí forever. Không cần thẻ tín dụng. 30 giây có URL live.
